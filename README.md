Анализ макроэкономических показателей для ритейл-компаний на основе данных ЦБ РФ.

## О проекте

Проект предоставляет инструменты для анализа влияния макроэкономических факторов (ключевая ставка, инфляция, курсы валют) на бизнес-модель ритейл-компаний.

### Ключевые возможности:
- **Сбор данных** с API ЦБ РФ в реальном времени
- **Анализ трендов** ключевой ставки и инфляции
- **Ритейл-аналитика** с учетом отраслевой специфики
- **Экспорт в БД** с готовыми SQL-запросами
- **Визуализация** в виде комплексных дашбордов
